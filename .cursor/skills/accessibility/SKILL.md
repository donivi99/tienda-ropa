# Accessibility

- Botones e inputs con labels visibles o `aria-label`
- Modales: `role="dialog"`, focus trap, Escape para cerrar
- Contraste mínimo WCAG AA en texto sobre fondos oscuros
- Formularios login/checkout navegables por teclado
